# Smart File Organizer

A simple Python script to organize your messy folders into clean sections like Images, Videos, and Documents.  
Just run it once, and your folder looks neat instantly.

## Usage
Clone this repo
```bash
    git clone https://github.com/ManoJ-46/smart-file-organizer.git
    cd smart-file-organizer
    python run.py
```
Enter the folder path when asked

Example
Before:
Downloads/
┣ photo1.jpg
┣ notes.txt
┗ movie.mp4

After:
Downloads/
┣ Images/photo1.jpg
┣ Documents/notes.txt
┗ Videos/movie.mp4
